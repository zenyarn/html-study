欢迎来到第一章：HTML 文档结构

这个文件演示了一个标准 HTML5 文档的骨架结构。

打开浏览器开发者工具（F12），在 Elements 面板可以看到 DOM 树。

DOM 树结构

浏览器解析这个文件后，会在内存中构建如下树形结构：

Document
└── html (lang="zh-CN")
    ├── head
    │   ├── meta (charset="UTF-8")
    │   ├── meta (name="viewport")
    │   ├── title
    │   └── meta (name="description")
    └── body
        ├── h1
        ├── p
        ├── p
        ├── hr
        ├── h2
        └── ...

在浏览器中验证

现在请你做一件事：

  1. 按 F12 打开开发者工具
  2. 切换到 Elements 标签页
  3. 观察左侧的 HTML 树形结构
  4. 尝试点击展开/折叠各个节点
  5. 把鼠标悬停在某个节点上，看页面上对应元素高亮

这个 Elements 面板显示的，就是浏览器解析你的 HTML 后生成的 DOM 树。

React 操作的"虚拟 DOM"，和这个结构在概念上完全一致。
