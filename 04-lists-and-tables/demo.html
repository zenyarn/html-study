术语词典

DOM
Document Object Model，文档对象模型。浏览器解析 HTML 后生成的树形数据结构。
SEO
Search Engine Optimization，搜索引擎优化。通过改善网站内容和结构，提升在搜索结果中的排名。
CLS
Cumulative Layout Shift，累积布局偏移。衡量页面视觉稳定性的 Core Web Vitals 指标之一。
